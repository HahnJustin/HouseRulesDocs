# Plugins For HouseRulesDocs
## Set up
To update the plugin table:
1. Copy the plugin name and repository URL or documentation URL in plugin column
2. Add a description
3. Add updated settings for all teammates to use

## Table of Plugins
Community Plugins can be found by going into *Settings -> Community Plugins -> Browse* and search for each plugin below. Add the plugin and configure if the settings column says so.
