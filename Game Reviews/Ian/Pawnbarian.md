# The Review - I liked it
The game itself is fun to play. I like the mechanics and found myself playing it because I wanted to. The game is the game and there is no menus you it feels straight into the game play. What I didn't like was that the UI or the look of the game is really lacking. The feel is there and the tool tips were there, but the board is just the board with a background or flair to make it stand out and enjoy it outside of the game itself. I do not think that this game is worth $10. More like $5 if I am feeling generous.

### What the game does well mechanically?
The pawnbarian as a piece that has abilities of other pieces was a good idea and concept. I like that there are a lot of different enemy pieces with fantasy "character" pieces. There is a lot of variety 

### What could it improve on? / What did I not like
Better UI and art. Could of used someone's help improve. The price is definitely not worth it.

### What the game does well in marketing?
I looks very standard on it's marketing. Steam page and has different OS to play it on. It seems like the dev is very plain in style and taste with website, steam page and game page. Has game reviews in quick blurbs. They have a [youtube video trailer.](https://youtu.be/IKr-AxVk7_8) It has good steam reviews. Perfect as a phone game which is a platform it supports.
